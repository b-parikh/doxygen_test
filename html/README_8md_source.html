5 This package shows how to document ROS packages with Python and C++ code.
6 
7 Within this package, there are two entities modeled as ROS nodes:
8 1. creator
9 2. requester
10 
11 The creator is tasked with creating regular polygons (polygons with equal side lengths). After
12 creating them, the creator stores these polygons. The requester asks the creator to create
13 polygons with certain specifications. At a set frequency, the creator will publish a list of all
14 the polygons that it has created so far. The requester reads this list and prints it out on
15 ROS_INFO.
